# Security Policy

## Supported Versions

Use this section to tell people about which versions of your project are
currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

To report a vulnerability, please email *support@turbotenant.com* with detailed information.

- We aim to respond within *72 hours*.
- If confirmed, we will coordinate a fix and credit responsible disclosure.
